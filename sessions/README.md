# AI Session Logs

For the assignment submission, export your AI coding session logs and place them in this folder.

## Cursor

- In Cursor, export the Composer/chat transcript (e.g., “Export Transcript”)
- Save it as: `ai_session_logs/cursor_export.md`

If you used other tools (Claude Code / Copilot / Windsurf / Continue.dev), add their transcripts here as well.
